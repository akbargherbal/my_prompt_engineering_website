Context: Framework Overview

The IPEV Loop is a structured protocol requiring AI agents to: 1. INTENT: State the objective 2. PLAN: Specify exact commands and parameters 3. EXECUTE: Run the planned commands 4. VERIFY: Confirm successful completion

Key framework features include: - Collaborative checkpointing (git commits + session saves) - Directive protocol for user interruption - Diagnostic mode for verification failures - Two-party system (User as operator, Agent as executor)
